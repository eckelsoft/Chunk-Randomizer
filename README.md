# Random Chunks (Chaos Update)

**Every chunk you visit is randomized to a new blocktype!**

**Explore the world..**

**.. defeat the ender dragon.**

![randomized chunks](https://cdn.modrinth.com/data/T6LCJpvP/images/05d5a9659534e2ee8218cde9ad52c6d43f8f7f30.png)

## Features
- **Lava & TNT**: Chunks can now be replaced entirely by **Lava** or **TNT blocks*** can now appear
- **Random Effects**: Visiting chunks now give you random effects from **Strength V** to **Instant Damage III**.
- **👾 Surprise Spawns** From a harmless flock of chickens to a world-ending **Wither** or a stealthy **Warden**.
- **Fluid Control:** Decide if oceans and lava lakes stay as they are or become part of the chaos..
- **Version 1.21.11 Features like Spear & Mace Meta, Copper**

## 🛠 Basic Commands

| Command | Description |
|:-------|:-------------|
| `/rc start` | Activate the challenge and start the timer. |
| `/rc stop` | Pause the madness. |
| `/rc reset` | Reset the timer to 00:00:00. |
| `/rc effects` | **Toggle:** potion efffects on and off |
| `/rc fluids <true/false>` | **Toggle:** Should Water, Lava and Fire be randomized? (Default: false) |
| `/rc spawnmonster max` | **Set maximal amount of surprise mobs** |
| `/rc spawnmonster chunk` | **Set amount of visit chunks to trigger surprise spawns** |
| `/rc debug` | **Toggle**: See exactly which block or entity is about to ruin your day. |

## 💡 Pro Tips & Customization

- **Control the Chaos**:
    - Toggle all status effects/curses using `/rc effects`.
    - Show or hide the action bar timer with `/rc timer`.
- **Disable Monster Spawns**:
  Set the chunk interval to a massive number if you just want to focus on the blocks:
    - `/rc spawnmonster chunks 99999`
- **Customize your UI**:
    - `/rc timer prefix [HC]`
    - `/rc timer suffix  survived!`

## Installation

1. Ensure **Fabric Loader** for **1.21.11** is installed.
2. Place the `.jar` in your `mods` folder.
3. Launch the game and run `/rc start` to begin the nightmare!
