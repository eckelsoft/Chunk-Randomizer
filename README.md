## 🛠 Basic Commands

| Command                   | Description                                                              |
|:--------------------------|:-------------------------------------------------------------------------|
| `/rc start`               | Activate the challenge.                                                  |
| `/rc stop`                | Pause the challenge.                                                     |
| `/rc effects`             | **Toggle potion effects** (Default: false)                               |
| `/rc fluids <true/false>` | **Toggle:** Should Water, Lava and Fire be randomized? (Default: false)  |
| `/rc exclude add <block>` | Add a block to the custom blacklist                                      |
| `/rc exclude clear`       | Reset your custom blacklist                                              |
| `/rc debug <0-3>`         | **Toggle** debug messages for protection, blocks, and effects.           |

## 💡 Pro Tips & Customization

- **Use /rc exclude add minecraft:...** to prevent a chunk from turning into a lag-fest.
- **Chunk Walker Boots**: They protect you from all world-randomization and effects, but they lose 10 durability per chunk.
- Status effects/curses are disabled by default. Enable or disable them using `/rc effects`. Active chunk effects are removed immediately when you leave the chunk, use Chunk Walker Boots, stop the challenge, or disable effects.
- Surprise monster spawns and the action bar timer have been removed.
